## Overview

The `@feedback/db` package provides the database layer for GTM Feedback, built on Drizzle ORM with PostgreSQL. It includes schema definitions, type-safe queries, and relation mappings for users, requests, feedback, areas, and Salesforce data.

## Installation

```bash
npm install @feedback/db
```

## Database Client Usage

### Basic Queries

<CodeGroup>
```typescript Select
import { db, requests, users } from '@feedback/db'
import { eq, desc } from 'drizzle-orm'

// Select all open requests
const openRequests = await db
  .select()
  .from(requests)
  .where(eq(requests.status, 'open'))
  .orderBy(desc(requests.updatedAt))

// Select user by email
const user = await db
  .select()
  .from(users)
  .where(eq(users.email, 'user@example.com'))
  .limit(1)
```

```typescript Insert
import { db, requests, feedback } from '@feedback/db'
import type { NewRequest, NewFeedback } from '@feedback/db/types'

// Insert a new request
const newRequest: NewRequest = {
  title: 'CSV Export Feature',
  description: 'Users need ability to export data to CSV',
  slug: 'csv-export-feature',
  status: 'open',
  creator: userId,
  areaIds: ['analytics', 'export']
}

const [request] = await db
  .insert(requests)
  .values(newRequest)
  .returning()

// Insert feedback
const newFeedback: NewFeedback = {
  requestId: request.id,
  accountId: 'sfdc_account_123',
  severity: 'high',
  description: 'Critical for Q1 2026',
  creator: userId,
  creationSource: 'manual'
}

await db.insert(feedback).values(newFeedback)
```

```typescript Update
import { db, requests } from '@feedback/db'
import { eq } from 'drizzle-orm'

// Update request status
await db
  .update(requests)
  .set({
    status: 'shipped',
    updatedAt: new Date().toISOString()
  })
  .where(eq(requests.id, requestId))

// Add follower to request
await db
  .update(requests)
  .set({
    followers: sql`array_append(followers, ${userId})`
  })
  .where(eq(requests.id, requestId))
```

```typescript Delete
import { db, feedback } from '@feedback/db'
import { eq } from 'drizzle-orm'

// Delete feedback (cascade deletes are configured)
await db
  .delete(feedback)
  .where(eq(feedback.id, feedbackId))
```
</CodeGroup>

### Relational Queries

<CodeGroup>
```typescript With Relations
import { db, users, requests, feedback } from '@feedback/db'
import { eq } from 'drizzle-orm'

// Query request with nested feedback and user data
const requestWithFeedback = await db.query.requests.findFirst({
  where: eq(requests.id, requestId),
  with: {
    feedback: {
      with: {
        creator: true
      }
    },
    creator: true
  }
})

// Query user with all their requests
const userWithRequests = await db.query.users.findFirst({
  where: eq(users.email, 'user@example.com'),
  with: {
    requests: {
      orderBy: desc(requests.updatedAt),
      limit: 10
    }
  }
})
```

```typescript Complex Queries
import { db, requests, feedback, sfdcAccounts } from '@feedback/db'
import { eq, desc, and, inArray, sql } from 'drizzle-orm'

// Get requests in specific areas with high-severity feedback
const urgentRequests = await db
  .select({
    request: requests,
    feedbackCount: sql<number>`count(${feedback.id})`,
    highSeverityCount: sql<number>`count(case when ${feedback.severity} = 'high' then 1 end)`
  })
  .from(requests)
  .leftJoin(feedback, eq(feedback.requestId, requests.id))
  .where(
    and(
      eq(requests.status, 'open'),
      sql`${requests.areaIds} && ARRAY['analytics', 'export']::varchar[]`
    )
  )
  .groupBy(requests.id)
  .having(sql`count(case when ${feedback.severity} = 'high' then 1 end) > 0`)
  .orderBy(desc(sql`count(${feedback.id})`))
```
</CodeGroup>

## Query Patterns

### Filtering by Arrays

```typescript
import { db, requests } from '@feedback/db'
import { sql } from 'drizzle-orm'

// Find requests in specific areas
const analyticsRequests = await db
  .select()
  .from(requests)
  .where(
    sql`${requests.areaIds} && ARRAY['analytics']::varchar[]`
  )

// Find requests with specific follower
const userFollowedRequests = await db
  .select()
  .from(requests)
  .where(
    sql`${userId} = ANY(${requests.followers})`
  )
```

### Aggregations

```typescript
import { db, feedback, requests } from '@feedback/db'
import { eq, sql } from 'drizzle-orm'

// Count feedback by severity for a request
const severityCounts = await db
  .select({
    severity: feedback.severity,
    count: sql<number>`count(*)`
  })
  .from(feedback)
  .where(eq(feedback.requestId, requestId))
  .groupBy(feedback.severity)
```

### Joins with Salesforce Data

```typescript
import { db, feedback, sfdcAccounts, sfdcOpportunities } from '@feedback/db'
import { eq } from 'drizzle-orm'

// Get feedback with account and opportunity details
const enrichedFeedback = await db
  .select({
    feedback,
    account: sfdcAccounts,
    opportunity: sfdcOpportunities
  })
  .from(feedback)
  .innerJoin(sfdcAccounts, eq(feedback.accountId, sfdcAccounts.id))
  .leftJoin(
    sfdcOpportunities,
    eq(feedback.opportunityId, sfdcOpportunities.id)
  )
  .where(eq(feedback.requestId, requestId))
```
